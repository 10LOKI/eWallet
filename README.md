PERSONAL-WALLET-APP
Personal Wallet App A modern PHP-based personal finance management application that helps users track expenses, set budgets, and manage their financial goals.

Features User Authentication: Secure registration and login system with CSRF protection

Budget Management: Set monthly budgets and track spending

Expense Tracking: Add, view, and delete expenses with categories

Dashboard: Real-time overview of budget vs expenses

Categories: Predefined expense categories (Food, Transport, Rent, Entertainment, Other)

Responsive Design: Modern UI with Tailwind CSS

Tech Stack Backend: PHP 8.1+ with OOP architecture

Database: MySQL 8.0

Frontend: HTML5, Tailwind CSS, JavaScript

Security: CSRF tokens, password hashing, input sanitization

Architecture: MVC pattern with custom routing
